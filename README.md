#  Personalized Learning Copilot

A **full-stack AI-powered web application** designed and developed manually to help students learn smarter through personalized study plans, intelligent assistance, and progress tracking.

---

## Project Overview

The Personalized Learning Copilot adapts learning content based on individual student progress. It helps learners plan their studies effectively, clear doubts instantly, and identify weak areas using analytics.

This project is **fully designed and implemented by the developer**, including frontend, backend, and AI integration logic.

---

## Key Features

*  Secure Student Login & Registration
*  Course Selection (e.g., DSP, Data Structures)
*  AI-powered Doubt Solving (Chat Interface)
*  Personalized Study Plans with deadlines
*  Topic-wise Quizzes
*  Progress Tracking & Performance Analytics
*  Faculty View (optional)

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript
* React.js (if applicable)

### Backend

* Node.js & Express.js / Flask / FastAPI
* RESTful APIs

### AI & Logic Layer

* Custom AI logic integrated using LLM-based APIs
* Rule-based and prompt-engineered responses

### Database

* MongoDB / MySQL / Firebase (as per implementation)

## System Workflow

1. User logs in and selects a course
2. System generates a personalized study plan
3. AI assistant helps with doubts and explanations
4. Quizzes evaluate understanding
5. Analytics module highlights weak areas
